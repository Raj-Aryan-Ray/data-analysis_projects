# XGBM and LGBM

This project focuses on advanced boosting algorithms: Extreme Gradient Boosting (XGBM) and Light Gradient Boosting Machine (LGBM) for machine learning prediction tasks.

Topics Covered:
- Boosting Algorithms
- XGBoost Basics
- LightGBM Basics
- Gradient Boosting
- Classification
- Feature Importance
- Hyperparameter Tuning
- Model Evaluation

Libraries Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost
- LightGBM

Objectives:
- Understand boosting techniques
- Build high-performance machine learning models
- Improve prediction accuracy
- Compare advanced ensemble algorithms

Key Learnings:
- Difference between bagging and boosting
- Working of gradient boosting algorithms
- Improving model efficiency and accuracy
- Feature importance analysis
