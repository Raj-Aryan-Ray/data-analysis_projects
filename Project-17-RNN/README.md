# Recurrent Neural Networks (RNN)

This project focuses on Recurrent Neural Networks (RNN), a deep learning technique designed for sequential and time-dependent data.

Topics Covered:
- Recurrent Neural Networks
- Sequential Data
- Hidden States
- Forward Propagation
- Backpropagation Through Time (BPTT)
- Text and Sequence Processing
- Time Series Prediction
- Deep Learning Basics

Libraries Used:
- Python
- Pandas
- NumPy
- TensorFlow
- Keras
- Matplotlib

Objectives:
- Understand sequential deep learning models
- Build RNN architectures
- Process sequence and text data
- Perform prediction on time-dependent datasets

Key Learnings:
- Working of recurrent neural networks
- Understanding sequence learning
- Importance of memory in neural networks
- Applications of RNN in NLP and time series
