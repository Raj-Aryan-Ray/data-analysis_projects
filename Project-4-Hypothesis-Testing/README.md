# Hypothesis Testing

This project covers the fundamentals of hypothesis testing and statistical inference using Python.

Topics Covered:
- Null Hypothesis (H0)
- Alternative Hypothesis (H1)
- p-value
- Significance Level
- One-Tailed Test
  
Libraries Used:
- Python
- NumPy
- Pandas
- SciPy
- Matplotlib

Objectives:
- Understand statistical testing methods
- Analyze sample data
- Make data-driven decisions
- Validate assumptions using statistical techniques

Key Learnings:
- Interpretation of p-values
- Decision making using hypothesis testing
- Understanding statistical significance
- Comparing datasets using tests
