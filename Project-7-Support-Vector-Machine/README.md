# Support Vector Machine (SVM)

This project focuses on Support Vector Machine (SVM), a supervised machine learning algorithm used for classification and prediction tasks.

Topics Covered:
- Supervised Learning
- Classification
- Hyperplane
- Support Vectors
- Kernel Functions
- Linear SVM
- Non-Linear SVM
- Model Evaluation

Libraries Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Objectives:
- Understand SVM concepts
- Build classification models
- Analyze decision boundaries
- Improve prediction accuracy

Key Learnings:
- Understanding support vectors and margins
- Importance of kernel functions
- Classification using machine learning
- Evaluating model performance
