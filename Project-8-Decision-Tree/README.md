# Decision Tree

This project focuses on the Decision Tree algorithm, a supervised machine learning technique used for classification and prediction.

Topics Covered:
- Decision Tree Basics
- Classification
- Tree Structure
- Root Node
- Leaf Node
- Splitting Criteria
- Gini Index
- Entropy
- Model Evaluation

Libraries Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Objectives:
- Understand decision tree concepts
- Build classification models
- Analyze feature importance
- Improve prediction accuracy

Key Learnings:
- Working of tree-based algorithms
- Understanding entropy and Gini index
- Data splitting techniques
- Evaluating machine learning models
